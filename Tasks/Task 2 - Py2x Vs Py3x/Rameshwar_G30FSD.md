
Difference Between Py2x and Py3x

Division Operator :-

Python 2x :-Return Integer.
Python 3x :- Return float.

Printing:-

Python 2x: - Print is Statement.
Python 3x- Print is function.

Input :-

Python 2x-Both input() and raw input() function.
Python 3x:- Only input() function.

Iteration:-

Python 2x:- Range () and XRange() function .
Python 3x:- Only range() function.

Way of Storing Character:-
 
Python2x:- Store in ASCII.
Python3x :- Store in Unicode value.
